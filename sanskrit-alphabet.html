梵语 Sanskrit

संस्कृतम् saṃskṛtam

संस्कृत 的字面意思是"神圣、净化、完美"，因此梵语被称为神圣的言语，是世界上最古老的语言之一，属于印欧语系的印度-伊朗语族。是印度古代宗教、哲学、文学和科学的主要用语。

天城文（देवनागरी devanāgarī）是书写梵语最常用的文字，也用于书写印地语hindī、马拉地语marāṭhī和尼泊尔语nepālī等语言。

国际梵文译音字母表 (IAST，International Alphabet of Sanskrit Transliteration)是为便于学术研究和国际交流，将梵语的天城文字母遵循国际标准转换为拉丁字母的过程。另外，ISO-15919 几个字母的转写区别：r̥ r̥̄ l̥ l̥̄ ē ō aṁ

所有声音都可以鼻音化。

1.1 元音 Vowels

स्वरः svaraḥ

元音是在发音过程中气流通过口腔而不受阻碍发出的音，也称为母音。

天城文从左至右书写，以一条横线(षिरोरेखा ṣirorekhā)连接每个字母

元音单独书写时同发音书面形式，如：अ आ इ ई उ ऊ ऋ ॠ ऌ ॡ ए ऐ ओ औ

短音与长音相似，只是有一些额外标记特征。

梵语中有16个元音：

5个短元音：अ इ उ ऋ ऌ 及其5个长元音：आ ई ऊ ॠ ॡ 其中 अ a इ i उ u 为主音

4个复合音：ए ऐ ओ औ

2个辅助音：अं अः

1.1.1 基本元音 Simple vowels

समानाक्षर samānākṣara

仅使用一个发音点发出的元音，称为基本元音或单元音。

अ a इ i उ u ऋ ṛ ऌ ḷ

当音长延长一倍时即成为长元音：आ ā ई ī ऊ ū ॠ ṝ ॡ ḹ

1.1.2 复合元音 Compound vowels

सन्ध्यक्षर sandhyakṣara

使用两个发音点发出的元音，称为复合元音或双元音，复合元音都是长音

ए e ऐ ai ओ o औ au

1.1.3 辅助元音 Support vowels

उभयक्स्हर ubhayakshara

两个在音节中的正常元音上添加的特定音。为便于示范发音，通常在辅助元音前加上一个基础元音 अ a

अं aṃ 称为随韵 After-sound

अनुस्वारः anusvāraḥ

符号：天城体写作 ं 转写为 ṃ 如：क + ं = कं (ka+ṃ=kaṃ)

表示前一元音的鼻音化，或根据后续辅音转化为相应的鼻辅音。在同组擦音前与前面的元音发同组鼻音；作尾音则发m

अः aḥ 称为止韵 Release

विसर्गः visargaḥ

符号：天城体写作 ः 转写为 ḥ 如：क + ः = कः (ka+ḥ=kaḥ)

清喉擦音呼气音，通常出现在词尾，发前一个元音的回声，如：aḥ发音aha，ūḥ发音ūhu，aiḥ发音aihi

1.2 辅音 Consonants

व्यञ्जनम् vyañjanam

辅音是在发音过程中气流通过口腔受阻碍发出的音，也称为子音

辅音独立书写形式：क ख ग घ ङ च छ ज झ ञ ट ठ ड ढ ण त थ द ध न प फ ब भ म य र ल व श ष स ह

为方便发音，辅音自带默认元音अ a 如：क 音译转写为 ka 而不是 k

若消除默认的元音，则使用消音标记(halant) ् 或称止韵符(विराम virāma) 如：क ka → क् k

梵语中有33个辅音，按发音气流可分3种：闭塞音sparśāḥ，中间音antaḥstha，擦音ūṣmāṇaḥ

梵语辅音可按发音部位(sthāna)和发音方式(prayatna)进行分类或分组(वर्ग varga)

按发音部位可分5类，对应口腔与舌位的不同位置：

  • 喉音 कण्ठ्य kaṇṭhya 或 ka-varga: ka kha ga gha ṅa ha
  • 鄂音 तालव्य tālavya 或 ca-varga: ca cha ja jha ña ya śa
  • 卷舌音 मूर्धन्य mūrdhanya 或 ṭa-varga: ṭa ṭha ḍa ḍha ṇa ra ṣa
  • 齿龈音 दन्त्य dantya 或 ta-varga: ta tha da dha na la sa
  • 唇音 ओष्ठ्य oṣṭhya 或 pa-varga: pa pha ba bha ma va

按发音方式可分为：

1.2.1 塞音 Plosive

स्पृष्ट spṛṣṭa

特点：气流完全阻塞突然释放

分类：

  • 清音 अघोषः aghoṣāḥ
  • 浊音 घोषवन्तः ghoṣavantaḥ

分别有送气महाप्राणाः mahāprāṇāḥ与不送气अल्पप्राणाः alpaprāṇāḥ区别

  • 不送气清音： ka ca ṭa ta pa
  • 送气清音： kha cha ṭha tha pha
  • 不送气浊音： ga ja ḍa da ba
  • 送气浊音： gha jha ḍha dha bha

1.2.2 鼻音 Nasal

आनुनासिकाः anunāsikāḥ

特点：气流通过鼻腔共鸣

ṅa ña ṇa na ma

例：ṇka

1.2.3 近音 Semivowel

अन्तःस्थ antaḥstha

特点：介开元音与辅音之间的滑音

ya ra la va

例：yoga

1.2.4 擦音 Sibilant

ऊष्माणः ūṣmāṇaḥ

特点：气流通过狭窄通道产生摩擦

śa ṣa sa ha

例：śiva

1.2.5 特殊辅音

  • 卷舌边音：ळ ḻa ळ्ह ḻha （见于吠陀梵语，古典梵语少用）
  • 止韵 विसर्गः visargaḥ 词尾呼气音
  • 随韵 अनुस्वारः anusvāraḥ 鼻化音

1.3 复合字母 Conjunct letters

संयुक्ताक्षर saṃyuktākṣara

复合字母是一组连续没有元音的辅音组合，通常与元音结合以形成完整的发音单位。

1.3.1 发音规则 Rules of pronunciation

同化 (Assimilation)

辅音在组合中可能改变发音以适应相邻辅音，如：

  • t + s → tsa 音变如英语cats中的ts
  • n + c → ñca 鼻音同化，n变为硬腭鼻音ñ

送气与浊化

送气辅音与不送气辅音结合时，送气可能减弱。

清辅音后接浊辅音可能浊化，如：t + da → dda

鼻音化

鼻音与塞音结合时，常伴随鼻腔共鸣，如 mpa

1.3.2 书写规则 Rules of writing

当有消音竖线辅音组合时，竖线消失，如：स्+त→स्त s+ta→sta / ष्+ट+ष्ट ṣ+ṭa→ṣṭa

当有消音右弧字母组合时，右弧消失一半，如：क्+न→क्न k+na→kna

特殊情况：र् r 在首位，以符号形式写在上面：र्+त→र्त r+ta→rta / र्+न→र्न r+na→rna

र ra 在次位，符号写在左下方，但圆形字母消音符保留：क्+र→क्र k+ra→kra / प्+र→प्र p+ra→pra / ट्+र→ट्र ṭ+ra→ṭra

根据辅音结合的位置和发音方式，可分为：

同部位辅音组合

发音部位相同的辅音结合，如：

  • त् + त → त्त t + ta → tta 双写辅音，延长塞音
  • mma 双唇鼻音 m 的延长

异部位辅音组合

发音部位不同的辅音结合，

  • 半元音+辅音：न् + च → न्च n + ca → yca
  • 鼻音+辅音：न् + त → न्त n + ta → nta

1.3.3 特殊复合辅音

क्ष (kṣa) त्र (tra) ज्ञ (jña) श्र (śra) 是梵语特有的高频复合辅音，需单独记忆。

  • क्(k) + ष(ṣa) → क्ष (kṣa)
  • त्(t) + र(ra) → त्र (tra)
  • ज्(j) + ञ(ña) → ज्ञ (jña)
  • श्(ś) + र(ra) → श्र (śra)

1.4 元辅音组合 Consonants with vowels

स्वरयुक्तव्यञ्जनानि svarayuktavyañjanāni

元辅音组合形式不同于独立书写，书面元音会以符号的形式有规律的附加在辅音的不同位置（在辅音后、前、下、上）

1.4.1 元音符号 Symbols

मत्र matra

अ आ ा इ ि ई ी उ ु ऊ ू ऋ ृ ॠ ॄ ऌ ॢ ॡ ॣ ए े ऐ ै ओ ो औ ौ

如：क ka；का kā；कि ki；की kī；कु ku；कू kū；के ke；कै kai；को ko；कौ kau

特殊情况：与r或ṛ结合的音节，如：रु ru रू rū रृ rṛ हृ hṛ हॄ hṝ

1.4.2 特殊标记 Special marks

鼻音化 आनुनासिक्यम् ānunāsikyam

月牙点（चन्द्रबिन्दु candra-bindu），标记为 ँ 转写作 m̐ 如：क + ँ = कँ (ka+m̐=kam̐)

半月形（अर्धचन्द्र ardha-candra），标记为 ॅ 转写作 ṁ 如：क + ॅ = कॅ (ka+ṁ=kaṁ)

标点符号 Punctuation

  • । 竖杠(दण्ड daṇḍa) 转写为：।和॥ 表示句子的结尾或诗句的中点；双竖杠表示段落或诗句的结束。
  • ऽ 略音符(अवग्रह avagraha) 转写为：' 用于分割复合词中的各个元素，表示元音因连读而被删除，长元音被删除，可以重复略音符。

如：

अथ योगानुशासनम्

atha yogā'nuśāsanaṃ

तदा ट्रष्टुः स्वरूपेऽवसथानम्

tadā draṣṭuḥ svarūpe'vasthānam

प्रत्यक्षानुमानागमाः प्रमाणानि

pratyakṣā'numānā''gamāḥ pramāṇāni

1.5 音节 Syllable

अक्षराणि akṣarāṇi

音节是梵语发音的基本单位，以元音分音节，每个音节只有一个元音。

  1. 音节的类型
    • 开音节：尾音为元音，如：क ka
    • 闭音节：尾音为辅音，如：म् m
  2. 音节的核心是元音

    每个音节必须包含一个元音，元音可以是独立的字母(वर्ण varṇa)，也可以与辅音结合。

    如：अ a 是一个独立的元音音节。क ka 是由辅音क् k和元音अ a组成的音节。

  3. 辅音与元音结合形成音节

    क् k + अ a= क ka; क् k+ इ i = कि ki; क् k + उ u= कु ku

  4. 两个或多个辅音结合在一起时，会形成一个复合辅音，并与一个元音组成音节

    如：क् k+ ष ṣ + अ a= क्ष kṣa

  5. 末尾有其他音将归于最后一个音节中

    如：सुखम् → सु खम् sukham → su kham

  6. 辅助元音与元音在同一个音节中

    如：संस्कृत → सं स्कृ त saṃskṛta → saṃ skṛ ta

  7. 带有长元音、辅助音、辅音组合的音节为重音节 गुरु guru；其他为轻音节 लघु laghu

    通常重音节发音时长为轻音节的两倍。

    符号：重- 轻⏑可轻可重⏓

1.6 连音 Junction

सन्धि sandhi

是指词与词之间或词内部音节之间的音变规则。这些规则是为了使发音更加流畅和自然。

梵语的连声规则非常复杂，涵盖了元音、辅音以及复合音的变化。

1.6.1 元音连音 vowel sandhi

当两个元音相连时，根据元音类型发生合并、替换或插入辅音，以实现发音流畅。

核心原则：

  • 优先合并：短元音常被长元音或复合元音取代
  • 简化发音：避免元音重复，通过音变减少发音断裂

连音规则：

同元音合并

两个相同元音相遇时合并为长元音

  • अ + अ → आ
  • a + a → ā
  • इ + इ → ई
  • i + i → ī
  • उ + उ → ऊ
  • u + u → ū
  • ऋ + ऋ → ॠ
  • ṛ + ṛ → ṝ
  • ऌ + ऌ → ॡ
  • ḷ + ḷ → ḹ

例句：

देव + आत्म → देवात्म

deva + ātma → devātma （神与灵魂 → 神灵）

अथ योग अनुशासनम् → अथ योगानुशासनम्

atha yoga anuśāsanaṃ → atha yogā'nuśāsanaṃ

异元音合并

不同元音相遇时，产生音变

  • (अ, आ) + (इ, ई) → ए
  • (a, ā) + (i, ī) → e
  • (अ, आ) + ए → ऐ
  • (a, ā) + e → ai
  • (अ, आ) + (उ, ऊ) → ओ
  • (a, ā) + (u, ū) → o
  • (अ, आ) + ओ → औ
  • (a, ā) + o → au
  • (अ, आ) + (ऋ, ॠ) → अर्
  • (a, ā) + (ṛ, ṝ) → ar
  • (अ, आ) + अर् → आर्
  • (a, ā) + ar → ār
  • (अ, आ) + ऌ → अल्
  • (a, ā) + ḷ → al
  • (अ, आ) + अल् → आल्
  • (a, ā) + al → āl
  • इ, ई + अ, आ → य, आय् अ
  • i, ī + a, ā → ya, āy a
  • उ, ऊ + अ, आ → व
  • u, ū + a, ā → va
  • ए, ऐ + अ, आ → अय, आय् अ
  • e, ai + a, ā → aya, āy a
  • ओ, औ + अ, आ → अव, आव् अ
  • o, au + a, ā → ava, āv a
  • ए + अ → अ + इ + अ → अय् अ
  • e + a → a + i + a → ay a
  • ओ + अ → अ + उ + अ → अव् अ
  • o + a → a + u + a → av a

例句：

पाद + उत्तानासन → पादोत्तानासन

pāda + uttāna → pādottānā

मरीचि + आसन → मरीच्यासन

marīci + āsana → marīcyāsana

复合元音拆分

复合元音 ऐai 和 औau 可能拆分为 अ a + इ i 或 अ a + उ u 进行连音

  • ऐ + अ → अ + अ + इ + अ → आय् अ
  • ai + a → a + a + i + a → āy a
  • औ + अ → अ + अ + उ + अ → आव् अ
  • au + a → a + a + u + a → āv a

长元音优先规则

若前后元音均为长元音，优先保留后元音

1.6.2 辅音连音 consonant sandhi

当辅音与辅音或元音相遇时，为消除发音断裂，根据发音部位、发音方式的协调性，发生同化、浊化、鼻化等音变。

核心原则：

  • 同化优先：前辅音被后辅音同化发音部位或清浊属性。
  • 简化发音：复杂辅音组合可能合并为复合辅音，如：क्(k) + ष(ṣa) → क्ष (kṣa)

连音规则：

同部位辅音同化

若前后辅音发音部位相同，前辅音被后辅音同化

  • त् + न = न्न
  • t + na = nna
  • तत् + नाम → तन्नाम
  • tat + nāma → tannāma
  • द् + न = न्न
  • d + na = nna
  • तद् + नाम → तन्नम
  • tad + nāma → tannāma
  • तत् + अपि → तद् अपि
  • tat + api → tad api （那也 → 即便如此）辅音连音 t → d 受后续元音影响浊化

清浊辅音同化

前辅音清浊属性与后辅音保持一致

tat + gacchati → tadgacchati

ud + tiṣṭhati → uttiṣṭhati

送气音弱化

送气辅音与其他辅音组合时，送气减弱或消失

path + ta → patta

鼻音化

辅音后接鼻音时，变为对应鼻音

鼻音后接塞辅音时，鼻音转化为对应发音部位的鼻辅音

sam + prayoga → samprayoga

ahaṃ + karomi → ahaṅkaromi

特殊复合辅音

高频复合辅音，需单独记忆其固定形式

  • क्(k) + ष(ṣa) → क्ष (kṣa)
  • त्(t) + र(ra) → त्र (tra)
  • ज्(j) + ञ(ña) → ज्ञ (jña)
  • श्(ś) + र(ra) → श्र (śra)

    • 复合辅音发音提示：对于高频复合辅音 ज्ञ (jña)，值得注意，在现代实际发音（尤其是在印地语和瑜伽术语中）中，它常被简化为 “gya” 的音。例如，单词 ज्ञान (jñāna，知识/智慧) 在实际中常读作 gyāna，单词 विज्ञान (vijñāna，科学) 在实际中常读作 vigyāna，单词 प्रज्ञा (prajñā，智慧) 在实际中常读作 pragyā，初学者了解这一点有助于理解实际的拼读。

1.6.3 止韵连音 visarga sandhi

当词尾的ḥ后接其他词时，根据后续音素发生音变，以实现发音流畅。

连音规则：

后接元音的音变

  • ḥ+短音a,i,u… → ḥ 变为对应元音的同部位擦音(o,e)
  • 如：शिवः + अहम् → शिवो'हम्
  • śivaḥ + aham → śivo'ham
  • ḥ+长音ā,ī,ū… → ḥ 变为对应元音
  • 如：तेजः + ईशः → तेज ईशः
  • tejaḥ + īśaḥ → teja īśaḥ

后接辅音的音变

  • ḥ+清辅音k,c,t,p… → ḥ 变为 ś, ṣ, s
  • 如：योगः चित्त → योगश् चित्त / मनः + तापः → मनस्तापः
  • yogaḥ citta → yogaś citta
  • मनः + तापः → मनस्तापः
  • manaḥ + tāpaḥ → manastāpaḥ
  • ḥ+浊辅音g,j,d,b… → ḥ 变为 r
  • 如：प्राणः + गच्छति → प्राणर् गच्छति
  • prāṇaḥ + gacchati → prāṇar gacchati
  • ḥ+鼻音ṅ,ñ,ṇ,n,m… → ḥ 脱落，鼻音延长
  • 如：शान्तिः + न → शान्तिं न
  • śāntiḥ + na → śāntiṃ na

句尾或停顿时的保留

ḥ后无词或需停顿，保持原音

如：

ॐ शान्तिः शान्तिः शान्तिः

oṃ śāntiḥ śāntiḥ śāntiḥ
